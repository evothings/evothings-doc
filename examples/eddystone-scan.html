Eddystone Scan

Scan For Eddystone. This app demonstrates how to scan for Eddystone Beacons.

Run this example in the Evothings Viewer app, or build a Cordova app with the plugin cordova-ble. iOS 7 or Android device with support for Bluetooth 4.0 and Android 4.3 or higher required.

Source Code

You can browse the source code for this example at the Evothings GitHub repository

The file index.html is the main HTML file of the app.

In file app.js you will find the JavaScript code for the app.

What You Need

You at least one Eddystone Beacon to use this app.

Run this example in the Evothings Viewer app on Android or iOS.

Optionally you can build a stand-alone native app using Cordova. You then need to include the Cordova plugin com.unarin.cordova.beacon. For further details see the Cordova build guide.

An iOS device (iOS 7 or 8) or an Android device with support for Bluetooth 4.0 (which includes BLE) is required. For Android version 4.3 or later is needed.

How To Get Up And Running

Follow these steps to get started with this example:

  • Run the Evothings Workbench on your desktop/laptop computer.
  • Run the Evothings Viewer app on an Android mobile device and connect to the Workbench.
  • Launch the example "Eddystone Scan" from the Workbench window by pressing "Run".
  • If everything works, the app should automatically detect your Eddystone Beacons.
